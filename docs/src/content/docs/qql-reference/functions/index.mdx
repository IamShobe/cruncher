QQL provides built-in functions for boolean logic, string manipulation, math operations, and conditional expressions. Functions can be used inside [`eval`](/qql-reference/commands/eval) and [`where`](/qql-reference/commands/where) commands.

## Boolean Functions

Boolean functions return `true` or `false`. They are commonly used in [`where`](/qql-reference/commands/where) filter expressions and in [`eval`](/qql-reference/commands/eval) to create boolean fields.

  { name: "isNotNull()", purpose: "Check if a value is not null", example: "isNotNull(user)", link: "/qql-reference/functions/booleans/isNotNull" },
]} />

Conditional functions return different values based on conditions. Use them in [`eval`](/qql-reference/commands/eval) to create derived fields with branching logic.
